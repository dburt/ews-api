= ews-api

Exchange Web Services API.

== Configuration

Set the endpoint

  EWS::Service.endpoint :uri => 'https://example.com/ews/exchange.asmx',
                        :version => 1         
 
Set the credentials if the service requires autentication. NTLM is used.

  EWS::Service.set_auth 'testuser', 'xxxxxx'
